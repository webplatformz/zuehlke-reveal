What is JavaScript?

EcmaScript / ES

The scripting language specification which is implemented by JavaScript. It is standardized by the organization Ecma International.

Ecma TC39 Committee

All new proposals are considered by the Ecma TC39 committee. They go through the following process before they are included in the Ecma Standard

  1. Proposal
  2. Draft
  3. Candidate
  4. Finished

Language Characteristics

  • Dynamic
  • Weakly Typed
  • "Truly" Object-Oriented
  • Interpreted (not compiled)

How & Where does it run?

The Browser

The most common use of JavaScript is to add client-side behavior to HTML pages: The browser interprets and runs the JavaScript code.

03 Objects

Objects

Remember, except for the 6 primitive types, everything in JavaScript is an Object.

Creating an Object

Object Literal

                    
                        const anObject = {};

Creating an Object

Object Literal

                    
                        const person = {
                            name: 'John',
                            age: 24
                        };

An object is basically a Map

Accessing Properties

Dot notation

                    
                        const john = {
                            name: 'John',
                            age: 24
                        };

                        john.lastName = 'Hero';

                        console.log(john.age); // 24
                        console.log(john.lastName); // Hero
                        console.log(john.middleName); // undefined

Accessing Properties

Bracket Notation

                    
                        const john = {
                            name: 'John',
                            age: 24
                        };

                        john['lastName'] = 'Hero';

                        console.log(john['age']); // 24
                        console.log(john['lastName']); // Hero
                        console.log(john['middleName']; // undefined
